HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The title element defines the title for a webpage.

CSS

  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Inline CSS: direcly in HTML file and is for quick and permanent changes
  • Internal CSS: enbeds entire stylesheet directly in the HTML file
  • External CSS: must be enbedded into HTML file via link in head
  • Separateion of Concerns: separating code into different files based on use
  • Selector: defines element or attributes
  • Declarations: Contain property and value
  • Content-Padding-Border-Margin
  • rel: specifies relationship between current document and linked document
  • href: specifies the location (URL) of external source derived from current location

Git

  • git status: checks what branch we are currently on
  • git checkout -b "branch-name": creats a new branch and switches to it
  • git add -A: add all changes to staging area
  • git commit -m "add starter code": makes snapshot of current code
  • git pull origin main: pull main branch from Github repo to local machine
  • git push origin "current branch": local changes are pushed to Github repo
